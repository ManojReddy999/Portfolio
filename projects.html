Developing a navigation framework using Graph Neural Networks (GNNs) and point cloud data for robust path planning in complex environments, with future integration of RGB data for semantic surface classification.

SLAM Projects

Visual-Inertial SLAM

Report GitHub

Implemented a robust Visual-Inertial SLAM system fusing IMU and stereo camera data via an Extended Kalman Filter (EKF) for pose estimation and landmark mapping.

The system addresses sensor noise and synchronization, achieving accurate trajectory estimation for autonomous navigation.

LiDAR-Based SLAM for Differential-Drive Robots

Report GitHub

Developed a comprehensive LiDAR-based SLAM system integrating multi-sensor odometry, 2D LiDAR scans, and RGBD data for a differential-drive robot.

Refined motion estimates via ICP scan matching, constructed occupancy grid maps, and employed pose graph optimization with loop closure using GTSAM for enhanced accuracy.

Motion Planning

Infinite-Horizon Stochastic Optimal Control

Report GitHub

Developed a control policy for a differential-drive robot for trajectory tracking and obstacle avoidance, formulated as an infinite-horizon stochastic optimal control problem.

Explored Certainty Equivalent Control (CEC) and Generalized Policy Iteration (GPI) to handle motion noise and uncertainties.

Motion Planning in 3D: A* & RRT Approaches

Report GitHub

Developed and compared search-based A* and sampling-based RRT* algorithms for collision-free path planning in continuous 3D environments with obstacles.

Implemented robust collision checking and evaluated trade-offs between path optimality and computational efficiency.

Dynamic Programming for Door & Key Navigation

Report GitHub

Developed a dynamic programming algorithm for an agent to navigate a Door & Key environment, minimizing energy costs while handling tasks like key pickup and door unlocking.

Modeled as an MDP, the algorithm computes optimal control policies for both fixed and random map configurations.
